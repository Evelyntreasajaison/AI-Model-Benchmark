🤖 **AI Model Benchmark Dashboard**

This project benchmarks the performance of two Large Language Models using a Streamlit dashboard.
It compares model responses based on inference speed, output quality, and usability using the Hugging Face Router API.
The goal is to understand which model performs better for different real-world AI applications such as chatbots, documentation generation, and AI assistants.

🚀 **Features**

1.Compare multiple LLMs with the same prompt
2.Measure inference speed
3.Evaluate response detail and output quality
4.Visualize results with interactive charts
5.View full model responses side-by-side
6.Easy to extend with additional models
